# Overview

EduPath is a full-stack developer career platform featuring AI mentoring, personalized learning roadmaps, and in-browser coding skill verification.

## Features

- **AI Mentoring**: Connects developers with AI-driven insights for career growth.
- **Code Editor**: Integrates Monaco Editor for in-browser skill assessments.
- **Data Visualization**: Uses Recharts to visualize user progress.
- **Export capabilities**: Allows users to export their resumes based on verified skills.

## Tech Stack
- Next.js 16
- React 19
- TypeScript
- Prisma
- PostgreSQL
- Groq/OpenRouter APIs
- Monaco Editor
- Recharts
- Vercel
