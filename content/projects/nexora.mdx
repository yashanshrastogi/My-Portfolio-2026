# Overview

Nexora is a voice-controlled OS interface enabling real-time automation of 40+ system commands via speech and text. It integrates offline STT (Speech-to-Text) and TTS (Text-to-Speech) pipelines.

## Architecture

Built with a multi-threaded architecture for low-latency, privacy-first performance. Because the LLM (Ollama) runs entirely locally, it guarantees complete privacy and works without an internet connection.

## Tech Stack
- Python
- Ollama
- STT/TTS Pipelines
- Multi-threading
