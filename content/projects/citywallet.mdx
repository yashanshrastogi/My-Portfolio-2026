# Overview

CityWallet is a hyper-personalized discount delivery platform. It integrates Groq LLM for intelligent offer matching based on user location, weather, and calendar context. Features real-time event streaming via Google Cloud Pub/Sub and Redis.

## Features

- **Context-Aware Offers**: Matches discounts based on real-time factors like weather and current location.
- **Fast Architecture**: Uses Redis and Google Cloud Pub/Sub for rapid event streaming.
- **AI Integration**: Powered by Groq LLM for intelligent decision-making.

## Tech Stack
- Next.js 14
- Django
- FastAPI
- TypeScript
- Groq LLM
- Google Cloud Pub/Sub
- Redis
- NextAuth 5
